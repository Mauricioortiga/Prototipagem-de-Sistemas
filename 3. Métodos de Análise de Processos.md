⚡Antes de criar protótipos, é importante entender bem os processos da organização que o sistema vai apoiar. 
A análise de processos mostra como o trabalho é realizado, quais são os problemas existentes e 
onde é possível melhorar, componentes valiosos para um desenvolvimento completo.
